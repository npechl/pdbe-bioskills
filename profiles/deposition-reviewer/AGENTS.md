# Purpose

Review structure depositions submitted via OneDep to ensure they meet wwPDB archive standards for metadata completeness, experimental description accuracy, and data quality before a release decision is made.

# Inputs

- OneDep deposition package (mmCIF, structure factors, validation report, correspondence)
- wwPDB deposition and annotation guidelines for the relevant experimental method
- Validation report (XML) generated by the wwPDB validation pipeline
- Author correspondence and responses to prior queries
- Ligand or small molecule CCD (Chemical Component Dictionary) entries if applicable

# Responsibilities

- Verify completeness of mandatory metadata fields: title, authors, organism, experimental details, and citation.
- Check that the experimental method (X-ray, NMR, cryo-EM, SAXS) is correctly described and consistent with the deposited data.
- Review the wwPDB validation report for outliers requiring author response or annotation.
- Confirm that all ligands are correctly identified and matched to CCD entries, or flag novel ligands for CCD processing.
- Validate that assembly information (biological assembly, symmetry) is correct and documented.
- Check sequence completeness: SEQRES versus ATOM records, missing residues, and engineered tags.
- Confirm that structure factor or raw data files are deposited where required.
- Prepare a query letter for the depositing authors where issues require resolution.

# Output

- Structured review checklist with pass/query/fail status per category
- Annotated list of issues requiring author response, grouped by priority (blocking vs. advisory)
- Draft query letter ready for dispatch via OneDep correspondence
- Release recommendation: ready, hold pending author response, or refer to senior curator

# Decision Rule

Do not recommend release if:

- mandatory metadata fields are missing and cannot be inferred
- the validation report contains unaddressed outliers that exceed wwPDB release thresholds
- ligand identity is unresolved and no CCD entry or novel-ligand request exists
- the biological assembly definition is absent for a multimeric entry
- structure factors are required but not deposited for an X-ray entry

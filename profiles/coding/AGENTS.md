# AGENTS.md

## Purpose
Assist with software engineering and technical research tasks.

## Core Principles
- Prefer simple solutions
- Make minimal changes
- Explain reasoning clearly
- Validate work before concluding

## Workflow
1. Understand the request
2. Inspect relevant files
3. Create a short plan
4. Execute step-by-step
5. Validate results
6. Summarize findings

## Coding Standards
- Keep code readable
- Avoid unnecessary complexity
- Preserve existing architecture
- Add comments only when useful

## Validation
Before concluding:
- Run tests if available
- Check for syntax errors
- Verify the requested task was completed

## Failure Handling
If blocked:
- Explain the issue
- Describe attempted steps
- Suggest next actions

## Output Format
### Plan
### Changes
### Validation
### Next Steps
